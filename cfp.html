REFinD Challenge

We invite participation for the Relation Extraction Challenge based on the REFinD dataset. The shared task is a part of the Fourth Workshop on Knowledge Discovery from Unstructured Data in Financial Services, co-located with SIGIR 2023. The competition is hosted on this Codalab Link .


Dates

  • Challenge Launch: April 1, 2023.
  • Training Data Release: Aril 1, 2023.
  • Public Test Set Release: May 12, 2023.
  • Private Test Set Release: June 2, 2023.
  • Final Submission to Leaderboard Deadline: June 19, 2023.
  • Result Announcement Date: June 26, 2023.
  • Paper Submission deadline: June 30, 2023.




Submission

  • Public Test: This phase evaluates algorithms on the REFinD public test set where the references are available. A submission needs to consist of results on the entire public test set to be considered as a valid submission.
    • The leaderboard for the public test set is hosted here on Codalab.
  • Private Test: This phase evaluates algorithms on the REFinD private test set with all references heldout. A submission needs to consist of results on the entire test set to be considered as a valid submission. This phase is aimed at the final evaluation of the model and one is not allowed to create multiple submissions using multiple teams.
The final evaluation of the model will be based on the results on the private test set. Please first submit your results to both leaderboards. To be eligible for result archives and consideration for awards, we kindly request you to send the following information to akshat.x.gupta@jpmorgan.com using your main contact email:
  • Team name.
  • Team members.
  • The username used in CodaLab submissions.
  • The forum link to your paper submission on Openreview and the link to your arxiv submission.
Dual Sumbission: We also allow submissions that are under review in other venues or have preprint versions. But please make sure to follow dual submission policies from other venues.
Non-Archival: The paper submission will be non-archival. The winners can present their work at the workshop.
Anonymity Period: We do not enforce an anonymity period. We allow preprint servers such as arXiv at any point of time.

Submission Format and Link: All submissions must be in PDF format and follow the current ACM two-column conference format, i.e. it should be no longer than 4 pages(including references, proofs, and appendixes). Submission will be accepted via Microsoft CMT.


Prizes

There will be one champion, one runner-up, and one honorable mention for this challenge. We will have awards for these winners.
